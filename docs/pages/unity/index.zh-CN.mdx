# Unity SDK 概述

Developerworks Unity SDK 是一个功能强大的开发工具包，专为 Unity 游戏引擎设计，提供了完整的 AI 能力集成。

## 主要特性

### AI 对话能力
- **NPC 客户端**: 创建智能 NPC 角色，支持自然语言交互
- **AI 聊天客户端**: 集成先进的对话系统
- **语音模块**: 支持语音识别和语音合成

### 服务集成
- **聊天服务**: 实时文本对话功能
- **转录服务**: 语音转文字服务
- **图像生成**: AI 图像生成能力

### 身份认证
- **本地令牌管理**: 安全的本地认证存储
- **认证流程管理**: 简化的认证流程
- **开发者密钥**: 支持开发者密钥认证

### 开发工具
- **Unity 编辑器集成**: 完整的编辑器工具支持
- **版本更新检查**: 自动检查 SDK 更新
- **示例场景**: 丰富的示例代码和场景

## 系统要求

- Unity 2021.3 或更高版本
- .NET Standard 2.1 支持
- 支持的平台：
  - Windows
  - macOS
  - WebGL
  - iOS
  - Android
