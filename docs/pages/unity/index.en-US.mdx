# Unity SDK Overview

Developerworks Unity SDK is a powerful development toolkit designed specifically for the Unity game engine, providing complete AI capability integration.

## Key Features

### AI Conversation Capabilities
- **NPC Client**: Create intelligent NPC characters with natural language interaction
- **AI Chat Client**: Integrate advanced conversation systems
- **Voice Module**: Support for speech recognition and text-to-speech

### Service Integration
- **Chat Service**: Real-time text conversation functionality
- **Transcription Service**: Speech-to-text service
- **Image Generation**: AI image generation capabilities

### Authentication
- **Local Token Management**: Secure local authentication storage
- **Auth Flow Management**: Simplified authentication flow
- **Developer Keys**: Support for developer key authentication

### Development Tools
- **Unity Editor Integration**: Complete editor tooling support
- **Version Update Check**: Automatic SDK update checking
- **Example Scenes**: Rich example code and scenes

## System Requirements

- Unity 2021.3 or higher
- .NET Standard 2.1 support
- Supported Platforms:
  - Windows
  - macOS
  - WebGL
  - iOS
  - Android
