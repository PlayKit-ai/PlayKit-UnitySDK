# 安装指南

学习如何在您的 Unity 项目中安装和设置 Developerworks Unity SDK。

## 系统要求

在安装 SDK 之前，确保您的开发环境满足以下要求：

- **Unity 版本**：2021.3 LTS 或更高版本
- **操作系统**：Windows、macOS 或 Linux
- **.NET 版本**：.NET Standard 2.1 或更高版本
- **包管理器**：Unity Package Manager (UPM)

## 安装方法

有几种方式可以安装 Developerworks Unity SDK：

### 方法 1：Unity Package Manager（推荐）

通过 Unity Package Manager 安装 SDK 是最简单的方式：

1. 打开您的 Unity 项目
2. 前往 **Window > Package Manager**
3. 点击左上角的 **+** 按钮
4. 选择 **Add package from git URL**
5. 输入 SDK 仓库 URL：
   ```
   https://github.com/your-org/developerworks-unity-sdk.git
   ```
6. 点击 **Add**

### 方法 2：手动安装

您也可以手动安装 SDK：

1. 从 [GitHub Releases 页面](https://github.com/your-org/developerworks-unity-sdk/releases)下载最新版本
2. 将包解压到项目的 `Packages` 文件夹
3. Unity 将自动检测并导入该包

### 方法 3：Asset Package

离线安装：

1. 从我们的发布页面下载 `.unitypackage` 文件
2. 在 Unity 中，前往 **Assets > Import Package > Custom Package**
3. 选择下载的 `.unitypackage` 文件
4. 点击 **Import** 将 SDK 添加到您的项目

## 验证安装

安装后，验证 SDK 是否正确设置：

1. 打开 Unity Console（**Window > General > Console**）
2. 查找类似这样的消息："Developerworks SDK v1.0.0 initialized successfully"
3. 检查 SDK 命名空间是否可用：

```csharp
using Developerworks.SDK;

public class TestScript : MonoBehaviour
{
    void Start()
    {
        Debug.Log("SDK 版本: " + DeveloperworksSDK.Version);
    }
}
```

## 配置

安装后，您需要配置 SDK：

1. 在 `Assets/Resources/DeveloperworksConfig.json` 创建配置文件
2. 添加您的 API 凭证：

```json
{
  "apiKey": "your-api-key",
  "apiSecret": "your-api-secret",
  "environment": "production"
}
```

<Callout type="warn">
  保护您的 API 凭证！切勿将它们提交到版本控制系统。
</Callout>

## 故障排除

### Package Manager 中未显示包

如果添加后包未出现：

1. 关闭并重新打开 Unity
2. 检查 Unity 控制台中的错误消息
3. 验证 git URL 是否正确
4. 确保您的系统已安装 git

### 编译错误

如果安装后出现编译错误：

1. 确保您的 Unity 版本满足最低要求
2. 检查所有依赖项是否已安装
3. 尝试重新导入包
4. 清除 Library 文件夹并让 Unity 重新构建

### API 密钥问题

如果您在使用 API 密钥时遇到问题：

1. 验证您的凭证是否正确
2. 检查配置文件是否在正确位置
3. 确保 JSON 格式有效
4. 如果需要重新生成凭证，请联系支持
