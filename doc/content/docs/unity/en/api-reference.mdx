# API Reference

Complete reference documentation for all classes, methods, and types in the Developerworks Unity SDK.

## Core Classes

### DeveloperworksSDK

The main SDK class for initialization and configuration.

#### Methods

##### Initialize

Initializes the SDK with configuration options.

```csharp
public static void Initialize(SDKConfig config)
```

**Parameters:**
- `config` (SDKConfig): Configuration object containing API credentials and settings

**Example:**
```csharp
DeveloperworksSDK.Initialize(new SDKConfig
{
    ApiKey = "your-api-key",
    ApiSecret = "your-api-secret",
    Environment = SDKEnvironment.Production
});
```

##### GetVersion

Returns the current SDK version.

```csharp
public static string GetVersion()
```

**Returns:** String representation of the SDK version

**Example:**
```csharp
string version = DeveloperworksSDK.GetVersion();
Debug.Log($"SDK Version: {version}");
```

#### Properties

##### IsInitialized

Gets whether the SDK has been initialized.

```csharp
public static bool IsInitialized { get; }
```

##### CurrentConfig

Gets the current SDK configuration.

```csharp
public static SDKConfig CurrentConfig { get; }
```

#### Events

##### OnConnectionStateChanged

Fired when the connection state changes.

```csharp
public static event Action<ConnectionState> OnConnectionStateChanged;
```

##### OnDataSynced

Fired when data is synchronized with the server.

```csharp
public static event Action OnDataSynced;
```

## Authentication

### DeveloperworksAuth

Handles user authentication and session management.

#### Methods

##### SignIn

Signs in a user with email and password.

```csharp
public static async Task<AuthResult> SignIn(string username, string password)
```

**Parameters:**
- `username` (string): User's email address
- `password` (string): User's password

**Returns:** AuthResult containing success status and user data or error

**Example:**
```csharp
var result = await DeveloperworksAuth.SignIn(
    "user@example.com",
    "secure-password"
);

if (result.Success)
{
    Debug.Log($"Signed in: {result.User.Email}");
}
```

##### SignInAnonymously

Signs in anonymously without credentials.

```csharp
public static async Task<AuthResult> SignInAnonymously()
```

**Returns:** AuthResult with anonymous user data

##### SignInWithProvider

Signs in using a third-party authentication provider.

```csharp
public static async Task<AuthResult> SignInWithProvider(AuthProvider provider)
```

**Parameters:**
- `provider` (AuthProvider): The authentication provider to use

**Supported Providers:**
- `AuthProvider.Google`
- `AuthProvider.Facebook`
- `AuthProvider.Twitter`
- `AuthProvider.Apple`
- `AuthProvider.Discord`

##### CreateUser

Creates a new user account.

```csharp
public static async Task<AuthResult> CreateUser(UserRegistration registration)
```

**Parameters:**
- `registration` (UserRegistration): Registration details including email, password, and display name

**Example:**
```csharp
var result = await DeveloperworksAuth.CreateUser(new UserRegistration
{
    Email = "newuser@example.com",
    Password = "secure-password",
    DisplayName = "New User",
    AcceptedTerms = true
});
```

##### SignOut

Signs out the current user.

```csharp
public static async Task SignOut()
```

##### SendPasswordResetEmail

Sends a password reset email to the specified address.

```csharp
public static async Task SendPasswordResetEmail(string email)
```

**Parameters:**
- `email` (string): Email address to send the reset link to

##### ChangePassword

Changes the current user's password.

```csharp
public static async Task ChangePassword(string currentPassword, string newPassword)
```

**Parameters:**
- `currentPassword` (string): Current password
- `newPassword` (string): New password

##### UpdateProfile

Updates the current user's profile.

```csharp
public static async Task UpdateProfile(ProfileUpdate update)
```

**Parameters:**
- `update` (ProfileUpdate): Profile updates to apply

**Example:**
```csharp
await DeveloperworksAuth.UpdateProfile(new ProfileUpdate
{
    DisplayName = "Updated Name",
    PhotoUrl = "https://example.com/photo.jpg"
});
```

##### SendEmailVerification

Sends an email verification to the current user.

```csharp
public static async Task SendEmailVerification()
```

##### RestoreSession

Restores a previously saved session.

```csharp
public static async Task<AuthResult> RestoreSession()
```

#### Properties

##### IsAuthenticated

Gets whether a user is currently authenticated.

```csharp
public static bool IsAuthenticated { get; }
```

##### CurrentUser

Gets the currently authenticated user.

```csharp
public static User CurrentUser { get; }
```

##### HasPersistedSession

Gets whether a persisted session exists.

```csharp
public static bool HasPersistedSession { get; }
```

#### Events

##### OnAuthStateChanged

Fired when authentication state changes.

```csharp
public static event Action<AuthState> OnAuthStateChanged;
```

##### OnTokenRefreshed

Fired when the authentication token is refreshed.

```csharp
public static event Action<string> OnTokenRefreshed;
```

##### OnSessionExpired

Fired when the user session expires.

```csharp
public static event Action OnSessionExpired;
```

## API Client

### DeveloperworksAPI

Makes API calls to the Developerworks backend.

#### Methods

##### GetUserData

Retrieves the current user's data.

```csharp
public static async Task<UserData> GetUserData()
```

**Returns:** UserData object containing user information

**Example:**
```csharp
var data = await DeveloperworksAPI.GetUserData();
Debug.Log($"User Level: {data.Level}");
```

##### UpdateUserData

Updates the current user's data.

```csharp
public static async Task<bool> UpdateUserData(UserDataUpdate update)
```

**Parameters:**
- `update` (UserDataUpdate): Data updates to apply

**Returns:** True if successful, false otherwise

**Example:**
```csharp
await DeveloperworksAPI.UpdateUserData(new UserDataUpdate
{
    Level = 10,
    Points = 1000
});
```

##### GetLeaderboard

Retrieves leaderboard data.

```csharp
public static async Task<Leaderboard> GetLeaderboard(string leaderboardId, int limit = 100)
```

**Parameters:**
- `leaderboardId` (string): ID of the leaderboard
- `limit` (int): Maximum number of entries to return (default: 100)

**Returns:** Leaderboard object with entries

##### SubmitScore

Submits a score to a leaderboard.

```csharp
public static async Task<LeaderboardEntry> SubmitScore(string leaderboardId, int score)
```

**Parameters:**
- `leaderboardId` (string): ID of the leaderboard
- `score` (int): Score to submit

**Returns:** LeaderboardEntry showing the player's position

##### GetAchievements

Retrieves the current user's achievements.

```csharp
public static async Task<Achievement[]> GetAchievements()
```

**Returns:** Array of Achievement objects

##### UnlockAchievement

Unlocks an achievement for the current user.

```csharp
public static async Task<bool> UnlockAchievement(string achievementId)
```

**Parameters:**
- `achievementId` (string): ID of the achievement to unlock

**Returns:** True if successfully unlocked

## Data Types

### SDKConfig

Configuration options for the SDK.

```csharp
public class SDKConfig
{
    public string ApiKey { get; set; }
    public string ApiSecret { get; set; }
    public SDKEnvironment Environment { get; set; }
    public bool EnableLogging { get; set; } = true;
    public TimeSpan Timeout { get; set; } = TimeSpan.FromSeconds(30);
}
```

### User

Represents an authenticated user.

```csharp
public class User
{
    public string Id { get; }
    public string Email { get; }
    public string DisplayName { get; }
    public string PhotoUrl { get; }
    public bool IsEmailVerified { get; }
    public DateTime CreatedAt { get; }
    public DateTime LastSignInAt { get; }
}
```

### AuthResult

Result of an authentication operation.

```csharp
public class AuthResult
{
    public bool Success { get; }
    public User User { get; }
    public AuthError Error { get; }
}
```

### UserData

User's game data.

```csharp
public class UserData
{
    public string UserId { get; }
    public int Level { get; set; }
    public int Points { get; set; }
    public Dictionary<string, object> CustomData { get; set; }
    public DateTime LastUpdated { get; }
}
```

### Achievement

Represents a game achievement.

```csharp
public class Achievement
{
    public string Id { get; }
    public string Name { get; }
    public string Description { get; }
    public int Points { get; }
    public bool IsUnlocked { get; }
    public DateTime UnlockedAt { get; }
}
```

### Leaderboard

Leaderboard data structure.

```csharp
public class Leaderboard
{
    public string Id { get; }
    public string Name { get; }
    public LeaderboardEntry[] Entries { get; }
    public int TotalEntries { get; }
}
```

### LeaderboardEntry

A single leaderboard entry.

```csharp
public class LeaderboardEntry
{
    public int Rank { get; }
    public string UserId { get; }
    public string DisplayName { get; }
    public int Score { get; }
    public DateTime SubmittedAt { get; }
}
```

## Exceptions

### SDKException

Base exception for SDK errors.

```csharp
public class SDKException : Exception
{
    public SDKErrorCode Code { get; }
    public SDKException(string message, SDKErrorCode code) : base(message) { }
}
```

### AuthException

Exception thrown by authentication operations.

```csharp
public class AuthException : SDKException
{
    public AuthErrorCode AuthCode { get; }
    public AuthException(string message, AuthErrorCode code) : base(message) { }
}
```

### APIException

Exception thrown by API operations.

```csharp
public class APIException : SDKException
{
    public int StatusCode { get; }
    public string ResponseBody { get; }
    public APIException(string message, int statusCode) : base(message) { }
}
```

### ValidationException

Exception thrown when validation fails.

```csharp
public class ValidationException : SDKException
{
    public string[] ValidationErrors { get; }
    public ValidationException(string message, string[] errors) : base(message) { }
}
```
